### Installation

__Using terminal:__
```bash

git clone https://github.com/ThomasNoSimp/App.git
cd App
java -jar Calculator.jar

```

__Downloading ZIP__

- Manually download the zip file of the repository.
- Extract it in the folder or directory.
- Complie the Java file.
- Run 'Calculator.jar'.
